## 简体中文

# HAMGIS 测亩 - 专业 GIS 土地测量应用

基于 Zepp OS 的专业地理信息系统（GIS）应用，专为智能手表设计，提供高精度的土地面积测量与管理功能。

### 核心功能
- **高精度测量**：实时 GPS 追踪，支持多点勾勒测量区域。
- **多单位转换**：支持亩、公顷、平方米等多种专业土地单位。
- **历史记录管理**：自动保存测量项目，支持查看详情、地图轨迹及导出。
- **地图视图**：直观展示测量路径与围合区域。
- **高度定制化**：可配置震动反馈、自动保存、屏幕常亮等设置。

### 安装与开发
1. 安装 [Zepp CLI](https://docs.zepp.com/docs/guides/tools/cli/)。
2. 克隆仓库。
3. 运行 `npm install` 安装依赖。
4. 使用 `npm run build` 进行编译，或 `npm run preview` 进行预览。

## English

# HAMGIS - Professional GIS Land Measurement App

A professional Geographic Information System (GIS) application based on Zepp OS, designed specifically for smartwatches to provide high-precision land area measurement and management.

### Key Features
- **High-Precision Measurement**: Real-time GPS tracking with support for multi-point area delineation.
- **Unit Conversion**: Supports various professional land units including Mu, Hectare, and Square Meter.
- **History Management**: Automatically saves measurement projects with support for viewing details, map tracks, and exporting.
- **Map View**: Intuitively displays measurement paths and enclosed areas.
- **Highly Customizable**: Configurable vibration feedback, auto-save, keep-screen-on, and more.

### Installation & Development
1. Install [Zepp CLI](https://docs.zepp.com/docs/guides/tools/cli/).
2. Clone the repository.
3. Run `npm install` to install dependencies.
4. Use `npm run build` to build, or `npm run preview` to preview.

## 日本語

# HAMGIS - プロフェッショナルGIS土地測量アプリ

Zepp OSベースのプロフェッショナルな地理情報システム（GIS）アプリケーションです。スマートウォッチ向けに設計され、高精度な土地面積の計測と管理機能を提供します。

### 主な機能
- **高精度計測**: リアルタイムGPSトラッキングにより、多点指定による計測エリアの作成に対応。
- **単位変換**: 畝（ムー）、ヘクタール、平方メートルなど、複数の専門的な土地単位をサポート。
- **履歴管理**: 計測プロジェクトを自動保存。詳細、地図上の軌跡の確認、およびエクスポートが可能。
- **マップ表示**: 計測経路と囲まれたエリアを直感的に表示。
- **高度なカスタマイズ**: 振動フィードバック、自動保存、画面常時点灯などの設定が可能。

### インストールと開発
1. [Zepp CLI](https://docs.zepp.com/docs/guides/tools/cli/) をインストールします。
2. リポジトリをクローンします。
3. `npm install` を実行して依存関係をインストールします。
4. `npm run build` でビルド、または `npm run preview` でプレビューを実行します。
